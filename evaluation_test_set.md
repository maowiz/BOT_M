# RAG Evaluation Test Set

Use this template to validate your chatbot's accuracy.
Fill in questions based on your crawled content.

## How to Use
1. After crawling a website, pick 10 questions whose answers ARE in the docs
2. For each, note the expected answer and expected source URL
3. Ask your bot each question and record results
4. Calculate: Retrieval Hit Rate = (questions where source was found) / 10
